Job Portal Web Application
=========================

A modern web application for job seekers and employers to connect and manage job postings, applications, and profiles.

Project Structure
----------------
The project consists of two main components:
- Frontend (FE_FINAL)
- Backend (JobPortal_WebApp)

Features
--------
- User authentication and authorization
- Job posting and management
- Job search and filtering
- User profiles for both job seekers and employers
- Application tracking system
- Responsive design for all devices

Technology Stack
---------------
Frontend:
- React.js
- Modern UI components
- Responsive design
- State management

Backend:
- Python
- Flask/Django (based on requirements.txt)
- Database management
- RESTful API

Getting Started
--------------
Prerequisites:
- Node.js (for frontend)
- Python 3.x (for backend)
- Virtual environment (recommended)

Installation:
1. Clone the repository
   git clone [repository-url]

2. Frontend Setup
   cd FE_FINAL
   npm install
   npm start

3. Backend Setup
   cd JobPortal_WebApp
   python -m venv myenv1
   source myenv1/bin/activate  # On Windows: myenv1\Scripts\activate
   pip install -r requirements.txt

Development
----------
- Frontend runs on: http://localhost:3000
- Backend runs on: http://localhost:5000
